# Changelog

## [1.0.0] - 2026-06-29

### Added
- Initial open-source release of LeronX Engine
- Script generation module (`leronx.script`)
- Scene planning and composition (`leronx.scenes`)
- Voice synthesis abstraction (`leronx.voice`)
- FFmpeg-based rendering pipeline (`leronx.render`)
- Subtitle generation and styling (`leronx.subtitles`)
- Stock footage asset matching (`leronx.assets`)
- Plugin system (`leronx.plugins`)
- Full pipeline orchestration (`leronx.pipeline`)
- 21 unit tests covering all modules
