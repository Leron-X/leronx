# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in LeronX Engine:

1. **DO NOT** open a public issue.
2. Email us at **security@leronx.org** with details and steps to reproduce.
3. You will receive a response within 48 hours.
4. If the vulnerability is confirmed, a patch will be released within 7 days.

## Scope

The open-source engine (this repository) is in scope. The following are **not** in scope:
- LeronX Pro SaaS (proprietary)
- Authentication / billing systems
- Desktop application

## Rewards

We do not offer monetary rewards, but contributors will be credited in release notes.
